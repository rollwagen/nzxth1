
## Scripts

**tools.sh** - downloads, (git-)clones, unzipps, brew-installs the the latest version of tools needed and references in the [Dortania](https://github.com/dortania) guide into this folder.

Tools downloaded:
* GenSMBIOS
* MountEFI
* ProperTree
* gibMacOS
* MaciASL
* OCConfigCompare
* USBMap
* CPUFriendFriend
* IORegistryExplorer
* SSDTTime

Tool(s) installed via 'brew':
* Hackintool
